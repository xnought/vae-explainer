Learn how Variational Autoencoders (VAE) work by visualizing one running in your browser.

- Paper: https://arxiv.org/abs/2409.09011

## Development

```bash
pnpm install
pnpm dev
```
which runs the hot-reloading dev server at http://localhost:5173
